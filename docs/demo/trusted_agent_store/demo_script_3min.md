# Trusted Agent Store — 最終デモ台本（3分）

---

## オープニング（〜20秒）

それでは、Agent Storeの評価パイプラインをデモします。
このストアでは、外部のAIエージェントを登録する際に**5段階の審査**を行い、ストアに登録することで、信頼できるエージェントだけをマッチングに通します。また、この評価パイプライン自体をOSSとして公開しているため、それでは悪意のあるエージェントを登録して、無事rejectされるか見てみましょう。

---

## Step 1: PreCheck（〜5秒）

まず**PreCheck**。Agent Cardと疎通し必須フィールドが揃っているかを検証します。

---

## Step 2: Security Gate（〜40秒）

次に**Security Gate**。ここが安全性の要です。

AISI（AI安全研究所）とAdvBenchで合計**約780件の攻撃プロンプト**があります。ここから「必須」レベルのものと残りは**危険度に応じた比率**でランダムサンプリングします。

評価では**ゼロトレランスキーワード** — システムプロンプトやAPIキーなど秘密情報の漏洩を示すワード — が応答に含まれていれば即ブロック。次にGeminiによる**セマンティック評価**で文脈を含めた合否を出します。

このステップでは、**ドメインに応じて独自の攻撃プロンプトをcsvに追加可能**で、足りていないドメインの網羅性を高めることができます。

---

## Step 3: Agent Card Accuracy（〜40秒）

続いて**Agent Card Accuracy**。エージェントの自己申告が本当かどうかを実動作で確認します。

Agent Cardの**useCaseやskillsフィールドから自動的に評価シナリオを生成**します。今回のデモではデータ収集エージェントを例にしているので、Agent Cardに記載の「顧客データ分析」や「レポート生成」というスキルから「顧客データを分析してレポートを生成してください」というシナリオが自動で作られます。

そのシナリオでA2Aプロトコル経由で実際にエージェントを動かし、Geminiが応答を**3軸**で採点します。**Task Completion**（タスク完了度）、**Dialogue Naturalness**（対話の自然さ）、**Information Gathering**（情報収集の効率性）です。各0.0〜1.0で、平均0.7以上がPASSになります。

さらに、エージェントが返すPDF・CSVなどの**Artifacts**も検出・解析して、ジャッジの評価材料に含めます。

---

## Step 4: Jury Judge（〜35秒）

そして**Jury Judge**。Step 2・3の結果を受けて、**ChatGPT・Claude・Geminiの3モデル**が「陪審員」として合議評価します。

評価の枠組みは**AISEV（日本AISIの安全性評価ガイド）の10観点**がベースです。これを**安全性（Safety）**、**タスク完了度（Task Completion）**、**ツールの適切な活用（Tool Usage）**、**自律性（Autonomy）**の**4軸**に集約しています。

各陪審員の意見が割れた場合はディスカッションで摺り合わせ、最終的に右上の裁判官エージェントがスコアを出し判決を下します。

---

## Step 5: Auto-Decision（〜10秒）

そして**Auto-Decision**。Trust Scoreが**90点以上なら自動承認、50点以下ならリジェクト、その間は人間レビュー**行きになります。

---

## W&B Weave によるトレーサビリティ（〜15秒）

ここまでの全ステップですが、エージェントとのやり取り — プロンプト・応答・Artifact交換 — は**W&B Weave**で全てトレースされています。エージェントハーネスとして機能し、評価結果の**再現性と透明性**を確保しています。

---

## クロージング（〜10秒）

最後に重要なのは、**この審査は登録時だけでは終わらない**という点です。
ストアでは仲介エージェントがマッチング後の利用フィードバックを収集し、**スコアが動的に変動**します。**使えば使うほど信頼性の精度が上がる仕組み**です。

また、この評価パイプライン自体をOSSとして公開しているため、開発者は自分の環境で同じ仕組みを動かして、公開前にエージェントを事前審査することもできます。

以上が、Trusted Agent Storeの評価パイプラインです。ありがとうございました。
